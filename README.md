# 2026 leetcode hot 100

### CI检查项
+ commit 格式检查
```text
feat：新增功能。通常用于添加新的功能或者功能改进。
fix：修复 Bug。用于修复代码中的错误或者 Bug。
docs：文档修改。用于更新文档、注释或者帮助信息。
style：代码样式修改。用于调整代码格式、空格、缩进等样式问题，不影响代码含义。
refactor：代码重构。用于重构代码结构、优化算法、调整变量名等，但不改变代码功能。
test：测试代码修改。用于添加、修改或者删除测试代码。
chore：构建或者工具相关的修改。用于更新构建工具、依赖项、脚本等，但不影响代码功能
```
