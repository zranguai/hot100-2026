动态规划: 状态转移方程，最优子结构，重叠子问题

定义f(i): 从0爬到i有多少种不同的方法
如果最后一步爬了 1 个台阶，那么我们得先爬到 i−1，要解决的问题缩小成：从 0 爬到 i−1 有多少种不同的方法
如果最后一步爬了 2 个台阶，那么我们得先爬到 i−2，要解决的问题缩小成：从 0 爬到 i−2 有多少种不同的方法

于这两种方法是互相独立的（爬的台阶个数不同），所以根据加法原理，从 0 爬到 i 的方法数等于这两种方法数之和
    状态转移方程: f(i) = f(i - 1) + f(i - 2)
递归边界: f(0) = 1 f(1) = 1 即: 从 0 爬到 0 有一种方法，即原地不动。从 0 爬到 1 有一种方法，即爬 1 个台阶
